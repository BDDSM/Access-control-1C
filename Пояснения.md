# Список пояснений и уточнений по использованию обработки:

- **Поиск по дереву метаданных** (с v4.6) — не платформенный, но работает как live search: вводите имя или синоним в поле над деревом, фильтрация выполняется при наборе (Enter не требуется). После паузы ~0,3 с дерево обновляется на сервере.
- **Отчёты по ролям** (с v4.6) — «Дополнительные функции → Отчеты по ролям»: два раздела на одной форме («Роли без пользователей», «Роли вне профилей»). Метрики объектов метаданных совпадают с отбором роли на главной форме; при нуле в регистре «Права ролей» выполняется доп. проверка через `ПравоДоступа`.
- **Поиск роли** в поле «Выберите роль» — фильтрация списка на клиенте при вводе имени или синонима.
- Обработка встраивается во внешние отчеты и обработки при необходимости.
